# python_projects

Early-stage Python projects demonstrating logic and algorithmic thinking
This repository contains some of my initial programming projects, developed to build a strong foundation in Python and problem-solving.
The projects focus on logical thinking, user interaction, and applying core programming concepts.

## Projects

### 1. Number Guesser
A program that guesses a number chosen by the user using a logical narrowing approach.

### 2. Wordle Game
A text-based implementation of the Wordle game with feedback on guesses.

### 3. Electronic Configuration Calculator
Calculates the electronic configuration of an element based on its atomic number.
